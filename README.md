# О приложении

Приложение для ведения личной коллекции фильмов. Позволяет отслеживать статус просмотра, отмечать любимые фильмы и быстро находить нужное.

# Возможности

- Поиск — фильтрация фильмов по названию
- Фильтры — просмотр фильмов по статусу:
    - Все фильмы
    - В планах (Planned)
    - Смотрю (Watching)
    - Просмотрено (Done)
    - Избранное (Favorite)
- Изменение статуса — переключение статуса по кругу:  
  `В планах → Смотрю → Просмотрено → В планах`
- Избранное — добавление/удаление фильмов в избранное
- Статистика — отображение количества:
    - Фильмов в статусе "Смотрю"
    - Просмотренных фильмов
    - Фильмов в избранном
